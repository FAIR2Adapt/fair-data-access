FAIR2Adapt Data Access

This service provides ODRL-based access control for FAIR2Adapt datasets. Data files are encrypted with AES-256-GCM and access is controlled via ODRL policies published as nanopublications.

How to request access

1. Set up a did:web identifier:
fair-data-access keygen && fair-data-access did-doc did:web:yourdomain.org public_key.pem
2. Open an access request on GitHub.
3. If approved, your wrapped decryption key will appear at:
https://fair2adapt.github.io/fair-data-access/keys/<did-hash>/<dataset>.key
4. Decrypt the data:
fair-data-access decrypt dataset.gpkg.enc -k <unwrapped-key>

Available datasets

  • hamburg-buildings — Building footprints with demographic indicators
  • hamburg-statistical-units — Statistical units with vulnerability indicators
  • hamburg-risk-private — Building-level flood risk outputs
